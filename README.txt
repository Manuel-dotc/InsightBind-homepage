InsightBind iframe-ready package
- InsightBind_iframe_ready.html : HTML file adjusted for iframe embedding.
- Includes compatibility CSS (backdrop-filter, gradient text) and a robust smooth-scroll script.
Usage:
- Host this file on a static host (GitHub Pages, S3) or drop into your server.
- Embed in Wix using an HTML iframe element: <iframe src="https://.../InsightBind_iframe_ready.html" style="width:100%;height:100vh;border:0"></iframe>
